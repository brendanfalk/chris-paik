# Chris Paik Blog

A minimalist blog for Chris Paik. 

https://chris-paik.vercel.app/

## How to add blog posts

Add a new <li></li> to `index.html` in the `#blog-posts` section.

## How to deploy

Host the files on any static site hosting provider (Vercel)
